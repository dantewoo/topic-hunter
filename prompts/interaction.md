# 交互层话术

## 开场

当用户触发选题 Skill 时，按以下流程与用户交互。

---

## 标准流程（3步）

### 第一步：选领域

向用户展示：

```
选一下你关注的领域（可多选，输入数字，如 1,3,5）：

1  科技/AI    — 新技术、产品发布、行业变动
2  商业/财经   — 公司动态、市场趋势、搞钱逻辑
3  社会/民生   — 热点事件、政策变化、群体现象
4  娱乐/文化   — 影视综艺、名人动态、流行梗
5  游戏/二次元  — 新游、赛事、圈层文化
6  健康/生活   — 养生、心理学、生活方式
7  知识/科普   — 冷知识、硬核解释、认知升级
8  混合/不限   — 全领域扫描，看什么能爆就推什么

直接回车默认选 8（不限）
```

用户输入规则：
- 输入数字（如 1,3,5）：选择对应领域
- 输入 8 或直接回车：混合/不限
- 输入无效内容：提示重新输入

### 第二步：选内容形态

向用户展示：

```
选一下你要做的内容形态（单选）：

1  短视频（抖音/快手）  — 14字标题，30秒-3分钟
2  图文（公众号/小红书） — 标题+封面逻辑
3  播客/长视频         — 深度内容，节奏不同

直接回车默认选 1（短视频）
```

用户输入规则：
- 输入 1-3：选择对应形态
- 直接回车：短视频
- 输入无效内容：提示重新输入

### 第三步：受众画像

向用户展示：

```
你的内容主要给谁看？（单选）

1  科技从业者/爱好者   — 追前沿，爱硬核，能看深度
2  创业者/投资人        — 要信息差，要判断依据
3  大学生/年轻群体      — 追热点，爱玩梗，注意力短
4  普通大众/下沉市场   — 共情驱动，要看得懂
5  知识型受众           — 认知升级驱动，愿意看完

直接回车默认选 4（普通大众）
```

用户输入规则：
- 输入 1-5：选择对应受众
- 直接回车：普通大众
- 输入无效内容：提示重新输入

---

## 特殊流程

### 快速模式（用户说"选题 快"）

跳过全部交互，使用默认值：
- 领域：8（不限）
- 内容形态：1（短视频）
- 受众画像：4（普通大众）

告知用户：
```
快速模式，使用默认配置：全领域 / 短视频 / 普通大众
开始搜索...
```

### 指定领域模式（用户说"选题 科技"）

跳过领域选择，使用用户指定的领域，其余走默认：
- 内容形态：1（短视频）
- 受众画像：4（普通大众）

告知用户：
```
指定领域：科技/AI
其余使用默认：短视频 / 普通大众
开始搜索...
```

### 老用户模式（有历史偏好）

```
上次你的偏好是：科技/AI + 短视频 + 科技从业者
沿用这个配置？直接回车确认，或输入"改"重新选择
```

用户输入规则：
- 直接回车：沿用上次偏好
- 输入"改"或其他内容：进入标准流程
